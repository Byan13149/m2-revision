Diffusion Models

Lectures: L19–L21 • Prince: Chapter 18

Overview

Diffusion models (DDPM) are currently the state-of-the-art for image generation. The core idea:

  1. Forward process: Gradually add Gaussian noise over $T$ steps until data becomes pure noise
  2. Reverse process: Learn a neural network to denoise one step at a time
  3. Sampling: Start from pure noise and iteratively denoise to generate data

Forward (Diffusion) Process

At each step $t$, add a small amount of Gaussian noise:

$$q(\mathbf{z}_t \mid \mathbf{z}_{t-1}) = \mathcal{N}(\mathbf{z}_t; \sqrt{1-\beta_t}\,\mathbf{z}_{t-1}, \beta_t \mathbf{I})$$

where $\{\beta_t\}_{t=1}^T$ is the noise schedule (variance schedule), typically $\beta_1 = 10^{-4}$ to $\beta_T = 0.02$.

Key Shortcut: Jump to Any Step

Define $\alpha_t = 1 - \beta_t$ and $\bar{\alpha}_t = \prod_{s=1}^{t} \alpha_s$. Then:

$$q(\mathbf{z}_t \mid \mathbf{x}) = \mathcal{N}(\mathbf{z}_t; \sqrt{\bar{\alpha}_t}\,\mathbf{x}, (1-\bar{\alpha}_t)\mathbf{I})$$
Interpretation: $\mathbf{z}_t$ is a mixture of the original data $\mathbf{x}$ and noise $\boldsymbol{\epsilon}$: $$\mathbf{z}_t = \sqrt{\bar{\alpha}_t}\,\mathbf{x} + \sqrt{1-\bar{\alpha}_t}\,\boldsymbol{\epsilon}, \quad \boldsymbol{\epsilon} \sim \mathcal{N}(\mathbf{0}, \mathbf{I})$$

Reverse (Denoising) Process

The reverse process is also Gaussian (for small $\beta_t$):

$$\Pr(\mathbf{z}_{t-1} \mid \mathbf{z}_t, \boldsymbol{\phi}) = \mathcal{N}(\mathbf{z}_{t-1}; \boldsymbol{\mu}_\phi(\mathbf{z}_t, t), \sigma_t^2 \mathbf{I})$$

A neural network (typically U-Net) predicts $\boldsymbol{\mu}_\phi$ (or equivalently, the noise $\boldsymbol{\epsilon}$).

Training: Simplified Loss

From ELBO to Simple Loss

The ELBO for diffusion models decomposes into a sum of KL divergence terms between pairs of consecutive steps $(t-1, t)$. After simplification, the training loss reduces to:

$$\mathcal{L}_\text{simple} = \mathbb{E}_{t, \mathbf{x}, \boldsymbol{\epsilon}}\left[\|\boldsymbol{\epsilon} - \boldsymbol{\epsilon}_\phi(\mathbf{z}_t, t)\|^2\right]$$
Training Algorithm
  1. Sample a data point $\mathbf{x}$ from the training set
  2. Sample a random timestep $t \sim \text{Uniform}\{1, \ldots, T\}$
  3. Sample noise $\boldsymbol{\epsilon} \sim \mathcal{N}(\mathbf{0}, \mathbf{I})$
  4. Compute noisy version: $\mathbf{z}_t = \sqrt{\bar{\alpha}_t}\,\mathbf{x} + \sqrt{1-\bar{\alpha}_t}\,\boldsymbol{\epsilon}$
  5. Train the network to predict $\boldsymbol{\epsilon}$ from $(\mathbf{z}_t, t)$: minimize $\|\boldsymbol{\epsilon} - \boldsymbol{\epsilon}_\phi(\mathbf{z}_t, t)\|^2$

ELBO Derivation (L21)

Graphical Model

Forward model (reverse diffusion): $\Pr(\mathbf{x}, \mathbf{z}_{1:T}) = \Pr(\mathbf{z}_T) \prod_{t=1}^{T} \Pr(\mathbf{z}_{t-1} \mid \mathbf{z}_t)$

Variational approximation (diffusion process): $q(\mathbf{z}_{1:T} \mid \mathbf{x}) = \prod_{t=1}^{T} q(\mathbf{z}_t \mid \mathbf{z}_{t-1})$

ELBO Decomposition

Apply the ELBO trick and use the causal structure of the graphical model to get terms that only involve consecutive timesteps $(t-1, t)$:

$$\text{ELBO} = \mathbb{E}_q\!\left[\log \Pr(\mathbf{x} \mid \mathbf{z}_1)\right] - \sum_{t=2}^{T} \mathbb{E}_q\!\left[D_\text{KL}(q(\mathbf{z}_{t-1} \mid \mathbf{z}_t, \mathbf{x}) \| \Pr(\mathbf{z}_{t-1} \mid \mathbf{z}_t))\right] - D_\text{KL}(q(\mathbf{z}_T \mid \mathbf{x}) \| \Pr(\mathbf{z}_T))$$

Each KL term is between two Gaussians ⇒ closed-form. The KL between two Gaussians $\mathcal{N}(\boldsymbol{\mu}_1, \boldsymbol{\Sigma}_1)$ and $\mathcal{N}(\boldsymbol{\mu}_2, \boldsymbol{\Sigma}_2)$ has an analytic formula.

Sampling

  1. Start: $\mathbf{z}_T \sim \mathcal{N}(\mathbf{0}, \mathbf{I})$
  2. For $t = T, T-1, \ldots, 1$: compute $\mathbf{z}_{t-1}$ using the learned reverse process
  3. Output: $\mathbf{z}_0 \approx \mathbf{x}$

Conditioning and Guidance

Classifier Guidance

Use a separate classifier $p(y \mid \mathbf{z}_t)$ to guide the diffusion process toward a desired class. Add the classifier gradient to the denoising step.

Classifier-Free Guidance

Train a single model that can be both conditional and unconditional. At sampling time, interpolate between the two:

$$\tilde{\boldsymbol{\epsilon}}_\phi = (1 + w) \boldsymbol{\epsilon}_\phi(\mathbf{z}_t, t, y) - w \cdot \boldsymbol{\epsilon}_\phi(\mathbf{z}_t, t, \varnothing)$$

where $w > 0$ controls guidance strength. Higher $w$ ⇒ more adherence to the conditioning (e.g., text prompt) but less diversity.

DDIM: Faster Sampling

DDPM requires $T$ steps (often 1000). DDIM (Denoising Diffusion Implicit Models) provides a deterministic, non-Markovian sampling process that can skip steps, allowing generation in far fewer steps (e.g., 50).

Applications

  • Text-to-image: DALL-E 2, Imagen, Stable Diffusion
  • Image editing: inpainting, super-resolution
  • Scientific: molecular generation, protein structure
