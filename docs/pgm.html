Probabilistic Graphical Models

Lecture: L16 (+ start of L17) • Source: Lecture notes only (not in Prince)

Core Idea

A PGM is a compact visual representation of a factorized joint distribution. It communicates causal/dependency assumptions and keeps you honest about what you're assuming about the world.

Directed PGMs (Bayesian Networks)

A directed PGM is a picture of a product of conditional probabilities:

$$\Pr(x_1, x_2, \ldots, x_K) = \prod_{k=1}^{K} \Pr\!\left(x_k \mid \text{parents}(X_k)\right)$$

Rules:

  • One factor per node: "node given its parents"
  • Arrows go from cause → effect
  • Arrows encode which variables each conditional is allowed to depend on

Example: Wet Grass

$R$ = rain, $S$ = sprinkler, $W$ = wet grass

$$\Pr(R, S, W) = \Pr(R) \cdot \Pr(S) \cdot \Pr(W \mid R, S)$$

Given $W=1$ (grass is wet), we can compute the posterior $\Pr(R, S \mid W=1)$ using Bayes' rule.

Two-Phase Workflow

  1. Describe the generative model: What causes what? What are the latent variables and parameters? Write down the forward probabilistic story.
  2. Decide how to compute: Exact conjugacy (tiny models), optimization (MLE/MAP), variational inference, MCMC, etc. These methods take the joint (or log joint) as input.
Common Mistake

Don't draw arrows from data to parameters! "$\mu$ computed from data" is an algorithm. The PGM must encode the generative story: parameters → data. Inference goes the other direction computationally, but the graph doesn't.

Probability Refresher

Key Rules

$$\text{Marginalisation: } \Pr(x) = \int \Pr(x, y) \, dy$$
$$\text{Conditional: } \Pr(a \mid b) = \frac{\Pr(a, b)}{\Pr(b)}$$
$$\text{Bayes: } \Pr(\omega \mid \mathcal{D}) = \frac{\Pr(\mathcal{D} \mid \omega) \, \Pr(\omega)}{\Pr(\mathcal{D})}$$

Bayes Rule: The Main Story of ML

TermRole
$\Pr(\omega)$Prior: what you know about parameters before seeing data
$\Pr(\mathcal{D} \mid \omega)$Likelihood: how well parameters explain the data
$\Pr(\omega \mid \mathcal{D})$Posterior: updated belief after seeing data
$\Pr(\mathcal{D})$Evidence: normalizing constant

MLE vs MAP

$$\hat{\omega}_\text{MLE} = \arg\max_\omega \Pr(\mathcal{D} \mid \omega)$$
$$\hat{\omega}_\text{MAP} = \arg\max_\omega \Pr(\mathcal{D} \mid \omega) \, \Pr(\omega)$$

MAP = MLE + prior. Adding a Gaussian prior ↔ L2 regularization. Adding a Laplace prior ↔ L1 regularization.

Example: Coin Flipping

Parameter $\omega$ (bias), observations $x_1, \ldots, x_N$ where $x_i \in \{0, 1\}$. Plate notation with conditionally independent flips:

$$\Pr(\omega, x_{1:N}) = \Pr(\omega) \prod_{n=1}^{N} \Pr(x_n \mid \omega)$$

Posterior via Bayes: $\Pr(\omega \mid x_{1:N}) \propto \Pr(\omega) \prod_{n=1}^N \Pr(x_n \mid \omega)$

Connection to Rest of Course

PGMs provide the foundation for understanding VAEs, diffusion models, and other latent variable models. The generative story (PGM) defines the joint distribution, and inference methods (variational inference, ELBO) are used when exact posterior computation is intractable.
